# Belajar Git

## Git Version
```
git --version
```
## Configure Git
```
git config --global user.name "widiarrohman1234"
git config --global user.email "widiarrohman1234@gmail.com"
```
## Initialize Git
```
git init
```
## Git status
```
git status
```
```
git status --short
```

## Git Staging Environment
### Add to staging environment
```
git add index.html
```
### Git Add More than One File
long command
```
git add --all
```
short command
```
git add -A
```

## Git Commit
```
git commit -m "first commit"
```
