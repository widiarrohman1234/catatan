# Dokumentasi pembelajaran Vue.Js
## 1. Instalasi Vue.js
Membuat package.json
```
npm init
```
Menginstall Vue.js
```
npm init vue@latest
```
Masuk ke dalam folder yang dibuat
```
cd sidafa-fe-vue
```
Menginstall NPM
```
npm install
```
Menjalankan Vue.js
```
npm run dev
```
Mem-build vue.js (jika terjadi perubahan pada .env)
```
npm run build
```
